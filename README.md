# 🛠️ Mi Taller de Código

¡Hola! Bienvenid@ a este repositorio. Aquí es donde guardo todos esos pequeños proyectos, scripts y experimentos que escribo de forma casual por puro entretenimiento o curiosidad.

> "El código es como el humor: si tienes que explicarlo, es malo." (Aunque aquí intentaré que se entienda).

## 🗂️ ¿Qué encontrarás aquí?

La idea es tener un poco de todo. La estructura suele ser:

*   **/Scripts**: Pequeñas utilidades para automatizar tareas.
*   **/Retos**: Soluciones a desafíos de lógica.
*   **/Playground**: Pruebas de nuevos lenguajes o librerías que voy descubriendo.
